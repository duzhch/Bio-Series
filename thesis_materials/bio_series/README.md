# Bio-Series Thesis Materials

This directory contains detailed documentation and resources for the "Bio-Series" experiments (BioMaster Framework) to support master's thesis writing.

## Contents

- **[bio_workflow_details.md](./bio_workflow_details.md)**: A comprehensive guide covering:
  - System Architecture (Wide & Deep Transformer).
  - Data Processing Pipeline (GWAS, Clumping, PCA).
  - Feature Engineering (PigBERT Delta, Gene2Vec).
  - Mathematical Formulations (Loss Functions, Model Components).
  - Software Dependencies and Tools.

## Usage

This material is structured to be directly referenceable for:
- **Methodology Section**: Detailed model architecture and mathematical formulas.
- **Data Processing Section**: Step-by-step workflow description.
- **Results Discussion**: Explaining the biological rationale (Priors) and error analysis (Loss functions).
